# VENZE

Łyżwy rentalowe i pingwinki do nauki jazdy — własna marka GAMAK. 64-95% taniej niż konkurencja. System V-Lock Buckles, wymienne wkładki Dry-Tech, stal hartowana Steel-HD.

## Co jest w tym folderze

- `dane/` - pliki kontekstowe marki (profil, persona, oferta, plan)
  - `profil.md` - opis marki i pozycjonowanie
  - `persona.md` - klient docelowy
  - `oferta.md` - modele, ceny, warunki
  - `plan.md` - roadmapa (szablon do wypełnienia)
- `materialy/` - materiały od dostawców (pusty — do uzupełnienia po decyzji inwestycyjnej)
- `venze_final.html` - strona produktowa (gotowa)
- `white logo.png` - logo marki
- `Gemini_Generated_Image_*.png` - grafika produktowa

## Jak aktualizować

- Nowe materiały od dostawców → wrzucaj do `materialy/`
- Zmiany w ofercie → aktualizuj `dane/oferta.md`
- Po uruchomieniu projektu → wypełnij `dane/plan.md`
- Status projektu → aktualizuj w `dane/projekty-status.md` (folder dane/ GAMAK)
