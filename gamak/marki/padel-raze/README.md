# PADEL RAZE

Korty padel pod klucz — własna marka GAMAK. Stal ocynkowana, szkło hartowane 12 mm, certyfikaty UDEM i TÜV Rheinland. Trzy modele: Classic, Pro, Panorama.

## Co jest w tym folderze

- `dane/` - pliki kontekstowe marki (profil, persona, oferta, plan)
  - `profil.md` - opis marki i pozycjonowanie
  - `persona.md` - klient docelowy (JST, kluby)
  - `oferta.md` - modele, ceny, warunki
  - `plan.md` - roadmapa i aktualny status
- `materialy/` - materiały od dostawców i dokumenty
  - `youngman/` - materiały dostawcy #1 (Youngman)
  - `art padel/` - materiały dostawcy #2 backup (Art Padel) + certyfikaty
  - `sstd padel/` - materiały dostawcy rezerwowego (SSTD)
  - `elite sports/`, `fortune group/`, `legend sports/` - inne oferty dostawców
  - `raporty deep reaserch/` - raporty rynkowe (producenci, rynek PL)
  - `katalog-padel-raze-2026.pdf` - katalog 10 stron A4 (bez cen, do druku)
  - `katalog-padel-raze-2026.html` - wersja HTML katalogu
  - `oferta-padel-raze-2026.html` - oferta handlowa
  - `nawierzchni betonowa - wariant A/B.pdf` - warianty podłoża
- `website/` - strona internetowa Padel Raze
  - `index.html` - strona produktowa (3 modele, bez cen)
  - `zdjecia/` - zdjęcia produktowe
- `backup/` - kopie zapasowe plików

## Jak aktualizować

- Nowe materiały od dostawców → wrzucaj do `materialy/[nazwa-dostawcy]/`
- Nowe raporty rynkowe → `materialy/raporty deep reaserch/`
- Zmiany w ofercie/cenach → aktualizuj `dane/oferta.md`
- Postępy projektu → aktualizuj `dane/plan.md`
- Status projektu → aktualizuj w `dane/projekty-status.md` (folder dane/ GAMAK)
