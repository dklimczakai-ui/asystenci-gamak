# NARZĘDZIA

Element projektu "Gabinet na Autopilocie". Narzędzia wewnętrzne do zarządzania leadami i metrykami sprzedaży.

## Co jest w tym folderze

- `crm-metryki.html` - CRM / dashboard metryk (wersja HTML, otwórz w przeglądarce)
- `crm-server.js` - backend serwera CRM (Node.js)

## Jak aktualizować

- Na start wystarczy arkusz Google Sheets (prostszy)
- CRM HTML jako docelowe narzędzie gdy będzie więcej leadów
- Status → aktualizuj w `dane/projekty-status.md`
