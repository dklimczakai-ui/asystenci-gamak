# GABINET AUTOPILOT — WEBSITE (PL)

Element projektu "Gabinet na Autopilocie". Strona sprzedażowa / landing page — wersja polska.

## Co jest w tym folderze

- `gabinet-autopilot-animated.html` - główna strona landing page (aktualna)
- `strona tekst.md` - tekst strony w markdown (do edycji treści)
- `blog/` - artykuły blogowe (SEO, content marketing)
- `img/` - grafiki i zdjęcia na stronę
- `polityka-prywatnosci.html` - strona prawna (RODO)
- `regulamin.html` - regulamin usługi

## Jak aktualizować

- Zmiana treści → edytuj `gabinet-autopilot-animated.html`
- Nowy artykuł blogowy → dodaj do `blog/`
- Nowe grafiki → dodaj do `img/`
- Zmiana ceny/oferty → zaktualizuj HTML + `dane/oferta.md`
- Status → aktualizuj w `dane/projekty-status.md`
